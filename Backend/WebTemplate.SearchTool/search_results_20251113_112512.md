# Search Results: .NET 9 breaking changes

## Quick Answer

.NET 9 introduces breaking changes like non-keyed parameter injection removal and default header value redaction in HttpClientFactory. Nullable properties in LINQ now always return empty when calling .ToString(). Consult Microsoft Learn for a comprehensive list of changes.
